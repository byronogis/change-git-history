# Change Git History

> A tool to change git history

## Prerequisites

- [git-filter-repo](https://github.com/newren/git-filter-repo.git)

## Usage

### Install Dependencies

```bash
pnpm i
```

### Adjust Config

update `config.js` to your needs

### Run

```bash
node change-git-history.js
```

## Change Fields

at present:
- user.name
- user.email
